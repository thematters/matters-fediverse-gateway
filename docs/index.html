/ Fediverse Gateway
Functional demo · 開發中

把 Matters 的長文，接上 Fediverse 這張聯邦網。

一座小而清楚的「大使館」，讓 Mastodon、Misskey、GoToSocial 的使用者讀到、追蹤、互動。

本頁連到一組可公開讀取的 ActivityPub prototype endpoints：WebFinger、actor、outbox、Article 與 NodeInfo。完整雙向 gateway runtime 仍在 G1 開發中；目前 demo 驗證的是公開長文如何被 discover 與表示。完整原始碼請見 GitHub repo。

這是什麼

Matters 是有十年歷史、超過 26 萬註冊使用者的中文長文平台。Fediverse（聯邦宇宙）則是一張開放的社交聯邦網，Mastodon、Misskey、Threads、GoToSocial 都是它的一部分。

這個 gateway 在兩者之間搭一座大使館：讓聯邦上的使用者能追蹤 Matters 作者、看到完整長文（以 Article 型別，不縮成微網誌摘要）、按愛心、留言、轉發。這些互動會回傳到 Matters 站內。同時，付費文、加密文、站內私訊不會外流。

Try the prototype

這組公開 endpoint 展示 Matters Fediverse Gateway 的 read-side federation surface。測試 actor 是 acct:alice@mashbean.github.io，長文以 ActivityPub Article 表示，而不是降級成短訊息。

   ◀─── 入站互動 ──────────────  │   gateway-core      │  ── 出站 Article ─▶

靜態 + 動態雙層：文章本身永遠在 IPFS 上，gateway 當機也讀得到；gateway 只負責「互動」這個動態部分。

核心技術決策

對外型別 = Article

不降級成 Note。長文得到完整 HTML、summary、attachment 的呈現空間，而不是被縮略成微網誌動態。

靜態 + inbox bridge

不做完整聯邦伺服器。文章本身走 IPFS，gateway 只承接互動，故障邊界清楚。

公開內容邊界程式化

付費 / 加密 / 私訊由 visibility gate 在程式碼層級擋住，不靠規範自律。

Container-first 部署

首發提供 reverse proxy / systemd / env 三件組，給小型自架者一條可重現路徑。
